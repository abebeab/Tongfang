Overview

Secure card terminals for non-biometric access control with card swiping and PIN code options

For users who wish to use non-biometric access control devices, Card Terminals are a great choice. Compact and well-designed, they support the combination of card swiping and PIN codes, providing the most basic yet reliably secure access control management for a long list of both indoor and outdoor applications.

Key Features

Advanced technology for comprehensive security management

Multiple Card Technologies

Support for MIFARE, DESFire, and 125kHz card technologies for flexible integration

High Security

Encrypted communication and secure authentication protocols protect your facility

Flexible Management

Standalone operation or integrated with centralized access control software

Advanced Features

Anti-passback, multi-card authentication, and time schedules for enhanced control

The Evolution of Access Control and Attendance

Authorized access has come a long way from using keys, pin codes, cards, and fingerprints. We now find ourselves stepping into the era of more advanced biometric recognition. The use of face recognition in access control and time attendance has been quite popular in the market, mostly for its advantageous "touch-free" experience.

With advanced deep learning algorithms and practical functions, Tongfang BMS Face Recognition Terminals improve security and efficiency in access control and time attendance, making them easier than ever!

Superior Recognition in Any Environment

Empowered by deep learning algorithms, Tongfang BMS Face Recognition Terminals can reach an amazing recognition speed of 0.2 seconds, with the accuracy rate of over 99%. Equipped with dual lenses, the terminals perform equally well under low or even zero-light environments. Therefore, the face recognition device can adapt to various indoor and outdoor attendance scenarios.

Top Level Security

The embedded face image anti-spoofing technology boosts security level at entry points. All user data is encrypted for effective information protection, and the optional privacy protection mode also secures users' personal information.

Easy to Manage and Operate

Tongfang BMS Face Recognition Terminals support quick and easy registration via multiple access methods, and convenient device configuration via web or GUI. Users can remotely check live view, unlock door, check event logs and more, via the mobile App on their smartphone.

  • Access methods include card, PIN code, fingerprint, face, and iris
  • Supports web or GUI configuration
  • Remote control via mobile App

Extensive Functionality via One Device

The face recognition device features extensive functionality with centralized management. Users can enjoy:

  • Flexible access control management with multiple authentication methods
  • Convenient video intercom via indoor station or mobile App
  • Remotely check live view via indoor station, mobile App, or centralized platform

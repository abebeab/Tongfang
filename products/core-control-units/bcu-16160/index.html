BL310 Embedded Computer

NXP i.MX6ULL Cortex-A7 Industrial ARM Controller

Cortex-A7 800MHz
512MB DDR3L
2x Ethernet Port

High-performance industrial computer designed for IoT edge computing. Rugged aluminum casing with flexible modular I/O expansion for RS485, CAN, and Analog inputs.

Power Supply24VDC ±10% (2A max)

Communication2x Ethernet (Switched), 1x RS-485
ProtocolsBACnet/IP, Modbus TCP/RTU, SNMP

Product Overview

The BCU-16160 sets a new standard for building automation controllers. Designed with integrators in mind, it features tool-free wiring terminals, LED status indicators for every I/O point, and a built-in web server for configuration without external software.

Industrial Reliability

Designed for harsh environments with -20°C to 70°C operation and IP40 protection.

High-Density I/O

32 points in compact design - 16 digital inputs and 16 analog outputs.

Easy Integration

Native BACnet/IP and Modbus TCP support for seamless BMS integration.

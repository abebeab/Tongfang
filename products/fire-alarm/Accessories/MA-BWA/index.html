The MA-BWA Waterproof Base provides robust environmental protection for fire detection equipment installed in challenging locations. With its IP65 rating, it offers complete protection against dust ingress and water jets, making it ideal for both indoor and outdoor applications where standard bases would be compromised.

Installation Steps
  1. Select mounting location with adequate clearance
  2. Ensure mounting surface is flat and clean
  3. Mount base using stainless steel fasteners
  4. Install conduit and seal entry point with grommet
  5. Connect wiring to terminal block
  6. Verify gasket is properly seated
  7. Install compatible detector head
  8. Test seal integrity and detector operation

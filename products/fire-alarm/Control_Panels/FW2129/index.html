Product Overview

The FW2129 Remote LED Annunciator offers unparalleled visibility for fire alarm system status. Designed to interface with Tongfang Maple Armor series fire alarm control panels, it replicates critical system indicators remotely, enabling security personnel to monitor zones without accessing the main panel.

Real-Time Monitoring

Instant LED feedback for Alarm, Supervisory, and Trouble conditions per zone.

Plug & Play

Simple two-wire connection reduces installation time and errors.

Ultra-Bright LEDs

Sunlight-readable indicators with adjustable intensity.

Key Features

  • UL 864 9th Edition Listed
  • Compatible with FW2105, FW2110 Series Panels
  • Surface or semi-flush mounting options
  • Remote silence and reset capabilities
  • Diagnostic LEDs for communication status
  • Adjustable brightness levels for day/night operation
Wall Mount
Standard surface mounting bracket included
Flush Mount
Trim ring for clean in-wall installation
Rack Mount
Optional 19" rack kit for control rooms

Installation guide: Use #12 screws for backbox, wire according to NFPA 72 standards. All terminals are labeled for zone mapping.

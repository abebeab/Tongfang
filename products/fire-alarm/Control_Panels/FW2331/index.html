Product Overview

The FW2331 NOU (Notification Output Unit) is an intelligent output expansion module designed to extend the notification appliance circuit capacity of Maple Armor fire alarm systems. It provides additional output circuits for audio and visual notification devices.

Dual NAC Circuits

Two independent notification appliance circuits with 1.5A capacity each.

Intelligent Control

Advanced microprocessor control with real-time monitoring and diagnostics.

Easy Installation

Simple DIN rail or panel mounting with clear terminal connections.

Notification Capabilities

Supports audio notification devices

Supports visual notification devices

Synchronized strobe capability

Temporal code 3 alarm pattern

March time pattern

System voice messaging

Test mode activation

Walk test support

Installation Guidelines
  • Mount in dry, protected location
  • Ensure proper ventilation
  • Follow local electrical codes
  • Use proper cable strain relief
  • Maintain minimum clearance around unit
Wiring Requirements
Power Wiring

14-16 AWG for 24VDC power. Separate power supply or FACP power.

Data Wiring

Shielded twisted pair for NOU communication (18-22 AWG).

NAC Wiring

12-18 AWG for notification circuits. Follow NEC requirements.

Configuration Steps
  1. Mount the NOU Output Unit in desired location
  2. Connect power supply (24VDC)
  3. Connect data communication cables to FACP
  4. Connect NAC circuits to notification devices
  5. Install end-of-line resistors as required
  6. Power up and verify communication
  7. Configure through FACP programming

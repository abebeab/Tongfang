Product Overview

The FW2522 Conventional Heat Detector is a dual-element heat detector designed for reliable fire detection in environments where smoke detectors are impractical or would cause nuisance alarms. It combines both fixed temperature and rate-of-rise detection technologies for comprehensive fire protection.

Dual Detection Technology

Combines fixed temperature and rate-of-rise detection for maximum reliability.

Flexible Wiring

Configurable for both 2-wire and 4-wire conventional fire alarm systems.

Industrial Grade

Designed for harsh environments where smoke detectors are not suitable.

Detection Technologies
Fixed Temperature

Activates when ambient temperature reaches 135°F (57°C). Factory calibrated for accuracy.

Rate-of-Rise

Detects rapid temperature increases of 15°F (8.3°C) per minute or more.

Intelligent Processing

Microprocessor-based analysis for reliable detection and reduced false alarms.

Installation Guidelines
  • Follow NFPA 72 spacing requirements (50' spacing typical)
  • Install in areas of expected heat accumulation
  • Avoid direct exposure to HVAC vents or drafts
  • Maintain minimum 4" clearance from walls and obstructions
  • Consider thermal lag in high ceiling applications
Wiring Configuration
2-Wire Configuration

Power and alarm signals on same pair. Maximum detectors per circuit: 20.

4-Wire Configuration

Separate power and alarm circuits. Allows remote annunciation.

Configuration Jumper

Internal jumper selects between 2-wire and 4-wire operation.

Installation Steps
  1. Select appropriate location based on NFPA 72 requirements
  2. Mount the heat detector base to ceiling or wall
  3. Connect wiring to terminal block (select 2-wire or 4-wire)
  4. Set configuration jumper if using 4-wire mode
  5. Install heat detector head onto base (twist-lock)
  6. Test detector using built-in test switch
  7. Verify alarm LED operation and system integration

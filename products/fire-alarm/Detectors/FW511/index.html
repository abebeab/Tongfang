Photoelectric Advantage: Photoelectric smoke detectors are most effective at detecting slow, smoldering fires that produce visible smoke particles. They provide earlier warning for these types of fires compared to ionization detectors and are less prone to nuisance alarms from cooking or steam.

Installation Guidelines
  • Follow NFPA 72 spacing requirements
  • Install on ceiling whenever possible
  • Avoid air vents, corners, and dead air spaces
  • Maintain minimum 4" clearance from walls
  • Test detector after installation
Wiring Requirements
Power Requirements

12-30VDC, 35μA standby, 50mA alarm. Use fire-rated cable.

2-Wire Installation

Single pair for power and alarm. Maximum 20 detectors per circuit.

Terminal Connections

Screw terminals accept 12-22 AWG wire. Polarity marked clearly.

Installation Steps
  1. Turn off power to fire alarm system
  2. Select location per NFPA 72 spacing requirements
  3. Mount detector base to ceiling or wall
  4. Connect wiring to terminal block (observe polarity)
  5. Install detector head onto base (twist-lock clockwise)
  6. Restore power to fire alarm system
  7. Test detector using magnetic test tool
  8. Verify alarm LED operation and system response

Product Overview

The FW2721 Dual Action Station is a critical component of any comprehensive fire alarm system, providing manual initiation capability for building occupants. Designed for intuitive operation in emergency situations, it combines immediate visual identification with reliable mechanical actuation. The dual-action design allows for either deliberate push-button activation or emergency break-glass operation, ensuring the device can be activated quickly and effectively regardless of the situation or user familiarity.

Manual Call Point Importance: Manual call points are essential components of any fire alarm system, providing building occupants with the ability to initiate an alarm when they discover a fire. Unlike automatic detectors, manual call points allow for immediate human response, which can be critical in situations where fires develop quickly or in areas not covered by automatic detection. The FW2721's dual-action design ensures it can be activated quickly and reliably by anyone, regardless of their familiarity with the device.

Installation & Configuration

Installation Guidelines
  • Install per NFPA 72 and local code requirements
  • Mount 48" above finished floor (ADA recommendation)
  • Place within 5 feet of exit doors
  • Ensure clear visibility and accessibility
  • Maximum 200 feet travel distance between units
Wiring Requirements
Electrical Requirements

24VDC system voltage, normally open contact.

Wiring Configuration

Two-wire connection for conventional systems.

Terminal Connections

Screw terminals accept 12-22 AWG wire. Clearly marked terminals.

Installation Steps
  1. Turn off power to fire alarm system
  2. Select mounting location per code requirements
  3. Mark and drill mounting holes
  4. Route wiring to installation location
  5. Connect wires to terminal block (NO contact)
  6. Mount call point to wall using provided hardware
  7. Test operation by activating push-button
  8. Verify alarm signal at control panel
  9. Reset device using provided key
  10. Install glass break element (if not pre-installed)

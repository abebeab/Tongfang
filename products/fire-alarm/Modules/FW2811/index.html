Product Overview

The FW2811 Input Module is an intelligent monitoring device designed for fire alarm systems. It monitors conventional input circuits and reports their status to the fire alarm control panel, providing essential monitoring capabilities for various building systems.

Installation Guidelines
  • Mount in dry, protected location
  • Ensure proper ventilation
  • Follow NEC and local codes
  • Use proper cable strain relief
  • Maintain minimum clearance around unit
Wiring Requirements
Power Wiring

18-22 AWG for 15-32VDC power from FACP or separate supply.

Data Wiring

Twisted pair for MAP communication (18-22 AWG).

Input Circuits

12-22 AWG for field wiring to monitored devices.

Configuration Steps
  1. Mount the Input Module on DIN rail or panel
  2. Connect power supply (15-32VDC)
  3. Connect data communication cables to FACP
  4. Connect input circuits to monitored devices
  5. Install end-of-line resistors on input circuits
  6. Power up and assign module address via FACP programming
  7. Test all monitored circuits

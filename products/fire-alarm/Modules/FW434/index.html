Installation & Configuration

Installation Guidelines
  • Install in accessible, well-ventilated location
  • Maintain minimum 36" clearance in front
  • Secure to wall or floor per local codes
  • Ensure proper grounding connection
  • Route cables through provided knockouts
Mounting Requirements
Space Requirements

Minimum 36" clearance in front for access and maintenance.

Mounting Hardware

Hardware included for wall or floor mounting. Concrete anchors sold separately.

Electrical Requirements

Ensure proper grounding via included bus bar. Follow NEC requirements.

Installation Steps
  1. Select installation location with proper clearance
  2. Assemble cabinet frame following provided instructions
  3. Mount DIN rails at desired heights
  4. Install cabinet on wall or floor using provided hardware
  5. Install grounding bus bar and connect to building ground
  6. Mount equipment components on DIN rails
  7. Route cables through knockouts and manage with included ties
  8. Connect equipment following manufacturer instructions
  9. Install door and adjust as needed
  10. Label all components and circuits

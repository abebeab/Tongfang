Product Overview

The FW801 Detector Base Module is engineered to provide a reliable, standardized mounting platform for fire detection devices in commercial and industrial fire alarm systems. Its universal 4" (100mm) mounting pattern ensures compatibility with virtually all smoke detectors, heat detectors, and multi-sensor devices on the market. The intelligent terminal design provides secure electrical connections while preventing incorrect installation through keyed connectors and polarity protection features. With UL 268 listing and compliance with NFPA 72 requirements, this base module delivers professional-grade performance for critical life safety applications.

Installation & Configuration

Installation Guidelines
  • Install per NFPA 72 and local code requirements
  • Mount on ceilings or side walls as required
  • Maintain required spacing from walls and obstructions
  • Use appropriate fasteners for mounting surface
  • Follow wire routing requirements per NEC
Wiring Requirements
Electrical Requirements

24VDC system voltage. Follow manufacturer's wiring diagram.

Wire Specifications

#12-22 AWG, 600V rated. Use fire alarm rated cable.

Terminal Connections

Screw terminals: 7-9 in-lbs torque. Spring clamp: insert fully.

Installation Steps
  1. Turn off power to fire alarm system
  2. Select mounting location per spacing requirements
  3. Mark mounting holes using base as template
  4. Drill holes and install appropriate fasteners
  5. Mount base to ceiling or wall
  6. Remove appropriate knockouts for wire entry
  7. Route fire alarm cable to base location
  8. Strip wire ends 3/8" (10mm) and connect to terminals
  9. Install detector onto base, aligning keyed connectors
  10. Test detector operation per manufacturer instructions

Product Overview

The FW2961 Horn Strobe is a combination audible/visual signaling device designed to meet modern fire alarm notification requirements. By combining a clear electronic horn with a high-intensity xenon strobe in a single unit, it provides comprehensive notification for all building occupants while minimizing installation time and complexity.

Dual Notification

Combines 85 dB horn with selectable 15/75/110 cd strobe for comprehensive alerting.

UL Listed Safety

Certified to UL1971 and UL464 standards for fire alarm signaling devices.

ADA Compliant

110 cd setting meets ADA requirements for sleeping area notification.

Dual-Mode Operation

Audible Mode: 85 dB electronic horn with Temporal Code 3 pattern

Visual Mode: Xenon strobe with clear white light output

Synchronization: Flash synchronized with horn pulses

Field Selectable: Candela setting and tone pattern

Energy Efficient: Low current draw for extended system capacity

Weather Resistant: IP54 rated for indoor/outdoor use

Combination Benefits: Using a single combination device instead of separate horn and strobe units reduces installation time by approximately 40%, decreases wiring requirements, and provides consistent spacing for both audible and visual notification. The FW2961 is ideal for modern building designs where aesthetics and installation efficiency are priorities.

Candela Selection: Proper candela selection is critical for ADA compliance and effective visual notification. Use 15 cd for general areas, 75 cd for areas with high ambient light, and 110 cd for sleeping areas as required by ADA. Always verify candela settings during installation and commissioning.

Installation Guidelines
  • Follow NEC, NFPA 72, and ADA requirements
  • Mount 80-96" (2.0-2.4m) above floor for wall mount
  • Ensure clear line-of-sight for strobe visibility
  • Avoid mounting near obstructions
  • Consider ambient light and noise levels
  • Set appropriate candela rating for area
Wiring & Configuration
Circuit Requirements

24VDC Notification Appliance Circuit (NAC).

DIP Switch Settings

Set candela rating (15/75/110 cd).

Load Calculation

Include 65mA per device in NAC calculations.

Spacing Requirements

Follow NFPA 72 spacing for strobe coverage.

Installation Steps
  1. Select mounting location per NFPA 72 and ADA requirements
  2. Set DIP switches for appropriate candela rating
  3. Mount back box (if used) or prepare surface mount
  4. Route wiring to device location in approved conduit
  5. Connect wires to screw terminals (observe polarity)
  6. Secure horn strobe to mounting surface
  7. Ensure device is level and securely mounted
  8. Verify electrical connections are tight
  9. Test device operation from fire alarm control panel
  10. Measure sound levels and verify strobe visibility
  11. Document installation and testing results
Strobe Spacing Guidelines
Sleeping Areas

110 cd, max spacing 50 ft (15m).

General Areas

15/75 cd, spacing per NFPA 72.

High Ambient Light

Use 75 cd, reduce spacing as needed.

Installation Guidelines
  • Follow NEC and local electrical codes
  • Ensure proper gasket seating for weatherproofing
  • Mount at recommended heights (NFPA 72)
  • Use proper torque on terminals (0.8 N·m)
  • Ensure adequate wiring clearance
  • Verify level installation for proper operation
Wiring Requirements
Power Wiring

14-22 AWG for notification appliance circuits.

Knockout Selection

Use appropriate knockout size for conduit.

Grounding

Properly ground according to NEC requirements.

Weatherproofing

Use conduit seals for outdoor installations.

Installation Steps
  1. Select mounting location per NFPA 72 requirements
  2. Install back box (if used) or prepare surface mount
  3. Remove appropriate knockouts for wiring entry
  4. Route and connect wiring to terminal screws
  5. Install EPDM gasket in base channel
  6. Mount base to wall/ceiling using provided screws
  7. Ensure base is level and secure
  8. Connect notification appliance to base wiring
  9. Align appliance with base and twist-lock into place
  10. Verify secure mounting and proper operation
  11. Test notification appliance function
Mounting Height Guidelines
Audible Signals

Mount 90" (2.3m) minimum above floor.

Visual Signals

Mount 80-96" (2.0-2.4m) for visibility.

Outdoor Mounting

Consider snow/ice accumulation levels.

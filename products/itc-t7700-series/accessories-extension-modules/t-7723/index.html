Product Overview – T-7723

The T-7723 is a compact 16-channel IP network fire alarm interface designed for smaller installations requiring fire alarm integration with ITC T-7700 series IP audio systems. In a space-saving 1U chassis, it provides all the essential features for voice evacuation and emergency notification.

16 Independent Inputs: Connect up to 16 zones from your fire alarm panel. Each input can be programmed with its own message, priority, and destination zones. Perfect for floor-by-floor alerts in smaller buildings.

USB MP3 Player: The front-panel USB port allows you to store emergency messages directly on a USB drive. Pre-record "Evacuate", "Shelter in Place", and other alerts. Messages can be assigned to specific inputs.

One-Key Alarm Cancel: In an emergency, authorized personnel can silence alarms with a single button press on the front panel. Configurable to require password or key switch for security.

Priority Override: Fire alarm inputs have the highest priority in the system, automatically overriding all other audio (background music, paging, etc.) to ensure emergency messages are heard.

1U Compact Design: Fits in any standard 19" rack, ideal for smaller equipment rooms and budget-conscious installations.

Typical Fire Alarm Integration Diagram

Fire Alarm Panel (Zone 1-16) → T-7723 → Network Switch → T-7700 Server → Zone Amplifiers/Speakers

Example: Fire alarm zone 3 triggers → T-7723 plays "Evacuate Floor 3" from USB → message broadcast to all speakers on floor 3.

Life Safety Note

The T-7723 is designed for use in fire alarm systems but must be installed in accordance with local codes and standards. Regular testing is required. Always consult with a licensed fire alarm contractor for system design and approval.

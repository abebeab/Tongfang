Wireless Remote Control (T-7720)

The T-7702M includes a wireless remote control suite (T-7720) for operation up to 1km away. Ideal for auditoriums, sports facilities, and large venues where the paging station is not within reach.

Remote Features:
  • Range: Up to 1km line-of-sight
  • Task Keys: 12 programmable keys for one-touch paging, music, or announcements
  • Function Keys: 6 keys for volume, mute, next/previous track
  • Interface: RS-232, baud rate 9600
  • Power: Battery operated (included)
  • Display: LCD screen for status feedback

The remote can be configured via T-7700RP software to trigger specific actions, zones, or pre-recorded messages.

App Overview – T-7700PA-iOS

The T-7700PA-iOS is the official mobile broadcasting application for Apple devices, designed for the ITC T-7700 Series IP network audio system. It delivers the full functionality of the broadcasting management system to iPhone and iPad, enabling facility managers, security personnel, and authorized staff to control public address and intercom systems from anywhere with network access.

Built with native iOS frameworks, the app provides a fluid, responsive touch interface that follows Apple's Human Interface Guidelines. It connects directly to T-7700RU series servers (or T-7700RP) via Wi-Fi or cellular data. Whether on an iPhone for quick announcements or an iPad for detailed terminal monitoring, T-7700PA-iOS delivers enterprise-grade broadcasting power in your pocket.

Native iOS Experience

Optimized for iPhone and iPad with split view, gesture controls, and support for iOS dark/light mode.

Siri Text-to-Speech

Leverages iOS built-in high-quality voices for natural-sounding announcements in multiple languages.

iPad Dashboard

Dedicated iPad layout with multi-column views for simultaneous terminal monitoring and zone control.

Ideal Use Cases

  • Hospital IT: Manage overhead paging from anywhere in the facility.

Key Features – T-7700PA-iOS

Broadcasting & Paging
  • ✓ Live paging: Real-time voice broadcast via iPhone/iPad microphone
  • ✓ Audio files: Play MP3, WAV, M4A from Music library or Files app
  • ✓ Text-to-speech: iOS native voices (multiple languages, Siri quality)
  • ✓ Zone control: Single, multiple, or all zones with visual selection
  • ✓ Emergency override: Highest priority broadcast with one tap
Intercom & Communication
  • ✓ Full-duplex intercom: Two-way audio with T-7702A, T-7700AN, T-7710
  • ✓ Call history: Recent intercom sessions with timestamps
  • ✓ Favorites: Quick-dial contacts for frequently called stations
  • ✓ Acoustic echo cancellation: Clear conversation in noisy environments
Monitoring & Control
  • ✓ Terminal dashboard: Real-time status, IP address, firmware version
  • ✓ Volume management: Individual/group volume adjustment with slider
  • ✓ Task viewer: Current and scheduled tasks, execution status
  • ✓ Alarm monitoring: Fire alarm zone status (read-only)
Task & Music Management
  • ✓ Create tasks: Select audio, zone, schedule, repetition
  • ✓ Edit/delete: Full task lifecycle management
  • ✓ Presets: One-tap activation of emergency or routine broadcasts
  • ✓ Upload audio: Add music files to server library from iOS device
iOS Integration
  • ✓ Dark/light mode: Follows system appearance
  • ✓ Split view: iPad multitasking support (Slide Over, Split View)
  • ✓ Face ID / Touch ID: Quick secure login (optional)
  • ✓ Push notifications: Optional alerts for system events
App Permissions

Microphone: Required for live paging and intercom. Local Network: Required to discover and connect to T-7700 server. Media Library: Optional, for accessing audio files. No personal data is collected or transmitted.

System Compatibility – T-7700PA-iOS

Server Compatibility
  • T-7700RU1 – Standard Server
  • T-7700RU2 – Advanced Server
  • T-7700RU3 – Ultimate AI Server
  • T-7700RP – Broadcasting Management Software
  • T-7700SR – Hot Backup Server (via primary)
  • Requires server software version 2.4.0 or higher
iOS Requirements
  • iOS 8.0 or later (iPhone, iPad, iPod touch)
  • Optimized for iOS 12, 13, 14, 15, 16, 17+
  • 100MB free storage
  • Wi-Fi or cellular data (4G/LTE/5G)
  • Apple ID for App Store download
Supported Terminals (Remote Control)
  • T-7702A / T-7702M Paging Microphones
  • T-7700A / T-7700AN Intercom Stations
  • T-7701 / T-7701Z / T-7704 Decoders
  • T-7700B Series IP Amplifiers
  • T-7705A / T-7707 Series IP Speakers
  • T-7782 / T-7784 Column Speakers
  • T-7710 / T-7731 Emergency Help Points
  • T-7772 Touch Panel (volume control)
Network & Security
  • TCP/UDP ports: configurable (default 8080, 8088)
  • VPN support: Works with iOS VPN configurations
  • Authentication: Server-validated credentials
  • Encryption: AES-256 for login
Licensing

T-7700PA-iOS requires a per-user license activated on the T-7700RU/RP server. Each licensed user can install the app on multiple iOS devices (iPhone, iPad) associated with the same Apple ID, with one active session at a time. Volume licensing via Apple Business Manager is available for enterprise deployments.

Cross-Platform Note

T-7700PA-iOS and T-7700PA Android are separate applications but share the same server-side license model. Users may switch platforms but require separate app downloads.
